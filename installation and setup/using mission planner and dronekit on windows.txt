steps:
1. On desktop, create a dir called Aviators. Inside aviators, create dir courseroot. Inside courseroot, create dir apm.
2. Open anaconda cmd prompt
3. run the command: conda create --name mydroneenv python=2.7
4. run the command: conda activate mydroneenv
5. run the command: pip install opencv-python==4.2.0.32
6. run the command: pip install pygame
7. run the command: pip install lxml
8. run the command: pip install pyyaml
9. run the command: conda install -c anaconda wxpython
10. run the command: pip install mavproxy
11. run the command: pip install dronekit
12. run the command: pip install dronekit-sitl
13. run the command: conda install -c anaconda git
14. git clone https://github.com/ardupilot/ardupilot.git
15. cd ardupilot
16. git submodule update --init --recursive
17. run the command: conda install -c anaconda pillow

For SITL:
1. in mission planner, go to 'simulation' tab, select the required model from the drop-down list. In our case, it is 'quad'. Now click on multirotor. This will load the required firmware.
2. Go to 'flight plan' tab, set the desired home location.
3. Open a terminal window, activate the env, cd to to the directory which has the code. 
4. python filename --connect tcp:127.0.0.1:5762.

For Hardware:
1. for connecting dronekit to the drone via telemetry:
vehicle = connect('com6',wait_ready=True,baud=57600)
for disconnecting:
vehicle.close()
